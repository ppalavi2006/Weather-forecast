# Weather-forecast
Weather Forecast App 🌦️
A responsive web application built using HTML, CSS, and JavaScript that provides real-time weather updates through a weather API. Users can search for any city and view current temperature, humidity, wind speed, and weather conditions in an easy-to-use interface.

## Features
- Search weather by city name
- Real-time weather updates
- Displays current temperature
- Shows humidity levels
- Displays wind speed
- Shows "Feels Like" temperature
- Weather condition description
- Sunrise and sunset timings
- Responsive and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
